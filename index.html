GIT COMMIT

La commande git commit permet de capturer un instatané des changements stagés du projet. Ces instantané sont des versions du projet.

GIT GC

Permet d'optimiser le dépôt en supprimant les fichiers inutiles

GIT remote

Pour se connecter à un dépôt existant.

Branche GIT

Répertorier, créer ou supprimer des branches.

GIT pull

Fusionner les modifications qui sont sur le dépôt distant

GIT tag

Marquer des commits spécifiques.

GIT log

Genere le log d'une branche.

GIT reset

Réinitialise l'index et répertoire de travail à l'état du dernier commit

GIT archive

Permet à son utilisateur de créer un fichier zip ou tar contenant les composants d'un arbre du dépôt.
